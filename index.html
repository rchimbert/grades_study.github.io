Project overview

This end-to-end project consists in cleaning and analyse some data from a kaggle dataset (The dataset). In this data project, I conducted an analysis of a dataset containing information about 395 students' grades in three Portuguese and math tests. The dataset included various student attributes, such as school type, gender, age, address, family size, parental education, parental occupation, reasons for schooling, guardian information, travel time to school, study time, the number of failures, and other personal factors. I used Python and the Pandas library to manipulate and analyze the dataset. During the analysis, I explored the relationships between these student attributes and their performance in math and Portuguese tests (G1, G2, and G3 grades). Based on my analysis, I identified several significant correlations and trends that affected student performance in these subjects. My project provides valuable insights into the factors that positively or negatively influence student grades in math and Portuguese. These insights can be used to make informed decisions and recommendations to improve student performance in these subjects. This study is detailed on this commented code available on my Github: The code.
